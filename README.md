# HR Analytics Dashboard

## Overview

Built an interactive HR Analytics dashboard using Power BI to analyze employee attrition, workforce demographics, salary distribution, and employee engagement.

## Tools

- Power BI
- DAX
- Power Query

## KPIs

- Total Employees
- Attrition Rate
- Average Salary
- Average Experience
- Job Satisfaction

## Dashboard Features

- Interactive slicers
- KPI Cards
- Dynamic visuals
- Employee insights
- HR reporting

## Key Insights

- R&D experiences the highest attrition.
- Overtime is associated with higher employee turnover.
- Male employees represent approximately 60% of the workforce.
- Life Sciences has the highest attrition among education fields.
